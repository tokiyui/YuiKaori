ゆいかおりニューシングル「Ring Ring Rainbow!!」発売記念＆Birthdayイベント開催決定！

Tweet

■日時
2015年8月23日（日）
＜第1部＞OPEN 13：00　START 14：00　
＜第2部＞OPEN 16：30　START 17：30
＊開場/開演時間は予定ですので、変更になる場合がございます。ご了承下さい。

 

■イベント内容
トーク＆ミニライブ(2曲)

 

■会場
都内某所

 

■応募締切
2015年8月10日（月）必着

 

■参加方法
・2015年8月5日発売、ゆいかおりニューシングル「Ring Ring Rainbow!!」＜初回限定盤＞（KICM-91605）を、対象店舗にて、ご予約（全額内金）、またはご購入頂きましたお客様に参加応募用紙をプレゼント。必要事項をご記入の上、ご応募下さい。

 

＊全額内金予約でイベント参加券お渡し後のキャンセルは不可となりますのでご注意下さい。
＊既に対象店舗にてご予約済みでイベント参加応募券をご希望の方は、ご予約の店舗にお問い合わせ下さい。
＊当日、会場にてご本人確認をさせて頂きます。
　ご当選者ご本人であることを証明できる顔写真付きの証明書（運転免許証・パスポート・写真付き住基カード・学生証など(コピー・期限切れ・社員証・taspo不可)）を必ずお持ち下さい。
　イベント会場へご入場頂く際、当選されたご本人である事の確認をさせて頂きます。お忘れになるとご入場できませんのでご注意下さい。
　ご本人確認ができない場合は、イベント参加をお断りさせて頂きます。

 

・応募用紙を葉書に貼ってご応募ください。
　葉書以外でのご応募（封書不可）、また必要事項に記入漏れのある方は無効となりますのでご注意下さい。
・応募の際に＜第1部＞＜第2部＞どちらかをお選び下さい。

 

■参加応募用紙配布対象店舗
・アニメイト全店
・ゲーマーズ全店
・とらのあな全店（一部店舗・通信販売は除く）
※オンラインショップ（通販）も対象のチェーンでは応募用紙配布締め切り等が店舗とは異なる場合がございます。
　詳細は各チェーンオンラインショップ（通販）をご確認下さい。
